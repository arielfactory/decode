Данное приложение полностью интерактивно. Изменение различных
параметров личности осуществляется как мышью и клавиатурой,
так и посредством графического интерфейса.
 
Чтобы изменить угол обзора, переместите курсор, удерживая клавишу «Shift».
Нажмите и удерживайте правую кнопку мыши при движении,
чтобы переместить курсор-бомбу.
  
Команды клавиатуры:
 
U : Включить / выключить грид
X : Пересчитать грид
L : Включить / выключить источники света
N : Включить / выключить обновление векторов отражения
C : Включить / выключить ручное управление курсором-бомбой
T : Выгрузить фрейм в высоком разрешении
H : Включить / выключить информационную панель
Пробел: Начать / остановить запись анимации
1 - 9 : Активировать ранее сохранённые параметры камеры
- / + : Удаление / приближение
